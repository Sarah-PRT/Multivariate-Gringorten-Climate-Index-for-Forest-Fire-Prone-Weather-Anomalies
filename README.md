# Multivariate Gringorten Climate Index for Forest Fire-Prone Weather Anomalies

This repository contains the source code for calculating a standardized multivariate climate index (Z-Score) based on Gringorten's empirical plotting position statistical method.
This script was adapted from Lanet et al., 2024 and developed to analyze the modification and intensification of climate hazards conducive to wildfire risk through the study of compound events.
It relies on the reference data file named 'Gringortten_MULTIVARIEindex.py
