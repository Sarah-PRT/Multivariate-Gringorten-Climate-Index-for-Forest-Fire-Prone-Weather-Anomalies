# Multivariate Gringorten Climate Index for Forest Fire-Prone Weather Anomalies

This repository contains the code for calculating a standardized multivariate climate index (Z-Score) based on Gringorten's empirical plotting position statistical method. Adapted from Lanet et al., (2024), this script captures the co-occurrence of climate factors to analyze how compound events modify and intensify wildfire hazards. The script defaults to evaluating three combined anomaly configurations: 1-, 3-, and 6-month timelines. The anomaly sign (positive or negative) is selected for each variable.
